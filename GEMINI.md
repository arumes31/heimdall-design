# Heimdall Redesign - Project Instructions

## Versioning
- **Master Engine Versioning**: Always increment the `master-theme` version number (found in `custom.css` and `custom.js`) when implementing new features or fixes.

## Memory & Privacy
- **Project Memory**: The `MEMORY.md` file and the `.gemini/` internal configuration folders are private to the developer and MUST NOT be committed to the repository.
